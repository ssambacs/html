CSS(Cascade Style Sheet)
--> 웹 문서에서 사용하는 세가지 언어가 있는데
	각 언어들 마다 사용하는 목적이 구분이 된다.
		HTML			- 문서의 구조를 정의
		CSS				- 문서를 보기좋게 꾸미는 역할
		JavaScript		- 문서에 동적인 동작을 부여하는 역할

	참고
		W3에서는 세가지 언어마다 각각의 문서를 따로 작성해서
		문서를 구성하기를 권장하고 있다.
		
HTML 문서에서 스타일을 적용시키는 방법

	1. Inline Style
		--> 태그에서 직접 스타일 속성을 부여해서 적용시키는 방법
			--> 코드길이가 길어지고
				유지관리에 불리해진다.
					
	2. Inner Style Sheet
		--> head 태그내에 style 태그를 만들고
			그 태그내에서 적용시킬 태그를 선택하고 
			필요한 속성과 속성값을 적용시키는 방법
			--> Inline Style 보다는 유지관리에 유리해진다.
				하지만 다른 문서에서는 해당 파일에서 만든 스타일을 가져다 사용할 수 없다.
			
	3. External Style Sheet
		--> 스타일 파일을 별도로 작성해서
			여러문서에서 적용시킬 수 있도록 하는 방법
			
			장점
				여러파일에서 동시에 같은 스타일을 적용시킬 수 있다.
				일관된 디자인 적용이 가능하다.
								
			적용방법
				
				head 태그내에 
				<link rel="stylesheet" type="text/css" href="파일위치">
